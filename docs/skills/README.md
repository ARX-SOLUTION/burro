## Common workflow

1. Read [../00-INDEX.md](../00-INDEX.md) and [../08-CONTEXT.md](../08-CONTEXT.md).
2. Load only the relevant skill and domain docs.
3. Locate existing source patterns before editing.
4. Make the smallest contract-safe change.
5. Run the narrowest validation command and report the exact result.
6. Update docs when behavior changes.
